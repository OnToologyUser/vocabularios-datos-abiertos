Abstract

Este vocabulario permite representar datos sobre la situación del tráfico en una ciudad. Se trata de un vocabulario que extiende la Semantic Sensor Network Ontology (W3C SSN Ontology), que se encuentra en la actualidad en proceso de estandarización, con el objetivo de hacerla compatible con herramientas de visualización ya existentes y con cualquier tipo de sistema que sepa cómo tratar con esta ontología. Se recomienda utilizar este vocabulario en conjunción con el vocabulario para representar el callejero de una ciudad, y más concretamente con la representación de tramos de vía, dado que los tramos de vía son normalmente los elementos que se observan cuando se ofrecen datos sobre tráfico en las ciudades. Específicamente, las clases y propiedades que contiene este vocabulario en la actualidad se centran en datos de intensidad de tráfico de tramos viales, pero puede ser extendida en el futuro a otros tipos de observaciones
